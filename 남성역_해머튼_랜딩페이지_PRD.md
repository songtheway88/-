# 남성역 해머튼 분양 홍보 랜딩페이지 PRD

## 프로젝트 개요

남성역 해머튼(구. 남성역 하이팰리스 4차) 지역주택조합 1차 조합원 모집을 위한 단일 페이지 홍보 랜딩페이지를 제작한다. 목표는 방문자가 페이지에 도착한 뒤 30초 이내에 핵심 가격 메시지를 인지하고, 상담 신청(전화 or 카카오) CTA까지 자연스럽게 유도되는 것이다.

---

## 기술 스펙

- **타입**: Single-page HTML (정적, 서버 불필요)
- **배포**: Netlify / GitHub Pages 정적 호스팅
- **반응형**: Mobile-first, 375px ~ 1280px 완전 대응
- **폰트**: Pretendard (CDN)
- **의존성**: 외부 JS 라이브러리 사용 금지 (바닐라 HTML/CSS/JS만 사용)
- **파일 구조**:
  ```
  index.html      ← 단일 파일로 완성 (CSS/JS 인라인 또는 동일 폴더)
  style.css
  main.js
  /images/        ← 이미지 폴더 (아래 목록 참고)
  ```

## 인터랙션 및 동작 명세

### 스크롤 애니메이션
- 섹션 진입 시 `opacity: 0 → 1` + `translateY(30px → 0)` 페이드업
- `IntersectionObserver` 사용 (라이브러리 없이 바닐라 JS)
- 딜레이: 카드가 여러 개면 100ms 간격으로 순차 등장

### 숫자 카운터 (Key Stats Bar)
- 뷰포트 진입 시 0에서 목표 숫자까지 카운트업 애니메이션
- 시간: 1.5초

### Q&A 아코디언
- 클릭 시 답변 `max-height` 트랜지션으로 부드럽게 펼침/접힘
- 동시에 하나만 열리도록

### 고정 CTA 바 (모바일 전용)
- 모바일에서 화면 하단에 `position: fixed`로 전화번호 + 상담 버튼 고정
- 히어로 섹션 위에서는 숨김, 스크롤 후 나타남

### 상단 고정 헤더
- 스크롤 50px 이상 시 반투명 다크 배경 전환 (transition 0.3s)
